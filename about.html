About Climate Monitoring System

This climate monitoring system collects real-time temperature and humidity data using a DHT11 sensor connected to an ESP32 microcontroller. The data is processed through a Python script and stored in a Google Sheet via Google Apps Script. The website then fetches this data for visualization.

Features include:

  • Dashboard: Real-time line charts showing temperature and humidity trends
  • History: Table-based view of all recorded data with date filtering
  • Live Readings: Thermostat-style display of the most recent measurements

Data is updated every 5 seconds, providing near real-time monitoring of environmental conditions.
