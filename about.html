About This Site

Welcome to the Climate Monitor, a project tracking temperature and humidity data in real-time. This site uses a sensor to collect data every 6 seconds, stored in a Google Sheet, and visualized with custom charts and Power BI integration.

  • Live Reading: Updates every 6 seconds with the latest temp and humidity from the sensor.
  • Chart: Features a line graph comparing temp and humidity trends, refreshing automatically.
  • History: Shows all recorded data, but updates only when manually refreshed via the sheet.
  • Predict: Forecasts the next 7 days based on historical trends, auto-loading on page visit.
  • Pie/Scatter: Displays temp and humidity distributions with a scatter plot for correlations, powered by Power BI.
  • Line Chart: Plots temp and humidity trends on one chart, updated via Power BI refreshes.
